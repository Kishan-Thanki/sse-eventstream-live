# Server-Sent Events (SSE) Live Stream

A lightweight demonstration of Server-Sent Events (SSE) using Go (`net/http`) and vanilla JavaScript (`EventSource`).

## Quick Start

```bash
go run main.go
```

Then open your browser to:
- [http://localhost:8080](http://localhost:8080) (Original Event Stream)
- [http://localhost:8080/search-app](http://localhost:8080/search-app) (Live Search)
